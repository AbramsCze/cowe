# Greyson CoWe Sample Application

Strongly recommended IDE is Intellij IDEA.

## Running projekt locally
```
	git clone 
	clean tomcat7:run
```

You can then access projekt here: http://localhost:8080/cowe/

## Database configuration
```
    jdbc.initDatabase=true
```
If database is autogenerated.

## Less compile

Via maven profile less-install.
